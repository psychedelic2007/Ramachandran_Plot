# Ramachandran_Plot

This Repository consists of python code to plot Ramachandran plot of any custom ".pdb" file.

## Running the command
-----------------------
In your terminal type the follwing command
```
python RamachandranPlotter.py --pdb /path_to_file/<file-name.pdb> 
```
